# Оферта & Смета — демо для ЛАЙМ

Конструктор коммерческих смет на основе оферты на оборудование (185 позиций, 14 блоков). Готовится как самостоятельное приложение, которое затем прикручивается к Битрикс24 в виде плейсмента карточки сделки или смарт-процесса.

## Что внутри

- Каталог из оферты ТМ-ЭФ 2026 (185 позиций) с поиском и фильтром по блокам.
- Конструктор сметы по разделам и залам: цена за день · количество · дней аренды · коэффициент.
- Индивидуальные цены клиента — переопределение базовой цены прямо в строке сметы и сохранение для будущих смет того же клиента.
- Расчёт итогов: «Без НДС», «УСН 5%», «НДС 22%».
- Экспорт в XLSX (сводная + лист по каждому залу) и JSON.
- Готовые шаблоны «Стандартная» и «Большая (3 зала + накладные)».
- Все данные локально в браузере (LocalStorage) — позже легко подменяется на REST к Б24.

## Стек

- Vite + React + TypeScript + Tailwind
- Zustand (state + persist)
- SheetJS (xlsx) для экспорта

## Локальный запуск

```bash
npm install
npm run dev
```

## Сборка под GitHub Pages

```bash
VITE_BASE=/oferta-smeta/ npm run build
```

Деплой настроен в `.github/workflows/pages.yml` — пуш в `main` собирает и публикует автоматически.

## Дальнейшие шаги (Б24-интеграция)

1. Завести в Б24 смарт-процесс «Сметы» с полями: клиент, мероприятие, период, состав (JSON), итоги.
2. Заменить `LocalStorage`-сторадж на REST-вызовы `crm.item.add/update/get` через REST-прокси (Vibecode / собственный бэкенд).
3. Поместить SPA в плейсмент `CRM_DEAL_DETAIL_TAB` карточки сделки и привязать смету к ID сделки.
